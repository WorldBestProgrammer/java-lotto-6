# 미션 - 로또
***
## 기능 구현 목록

### View
***
- 입력
  - [ ] 입력 받기
- 출력
  - [ ] 구입금액 입력 부분을 출력
  - [ ] 구입한 개수와 로또 번호를 출력
  - [ ] 당첨 번호 입력 부분을 출력
  - [ ] 보너스 번호 입력 부분을 출력
  - [ ] 당첨 통계 출력
### Model
***
- 저장
  - [ ] 사용자가 산 모든 로또
### Controller
***
- 클래스
  - [ ] 메인(로또 게임 진행)
  - [ ] 입력값 처리

    - [ ] 당첨 번호 입력
    - [ ] 보너스 번호 입력
  - [ ] 로또 컨트롤러
    - [ ] 로또 번호 발행
    - [ ] 로또 당첨 내역 만들기
